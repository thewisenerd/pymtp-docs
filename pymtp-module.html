Module pymtp

source code

PyMTP is a pythonic wrapper around libmtp, making it a bit more friendly to use in python

Example Usage (or see examples/):

>>> import pymtp
>>> mtp = pymtp.MTP()
>>> mtp.connect()
PTP: Opening session
>>> print mtp.get_devicename()
Device name
>>> mtp.disconnect()
PTP: Closing session
>>>
